Step 2 · Install ESP Flash Tool (ESPDroid)
1
Install ESPDroid from Play Store
Search "ESPDroid" on Google Play, or use:
▶ Open Play Store — ESPDroid
2
Connect ESP32 via USB OTG
Plug your ESP32 into your phone using a USB OTG adapter (USB-A female to USB-C/micro). When prompted, allow USB access for ESPDroid.
3
Configure flash addresses
In ESPDroid tap Add file three times and set each one:
0x1000 → nRFBox_V2.ino.bootloader.bin 0x8000 → nRFBox_V2.ino.partitions.bin 0x10000 → nRFBox_V2.ino.bin
4
Set baud rate and flash
Set baud to 460800 (or 115200 if it fails). Tap Flash. Hold the BOOT button on ESP32 if flashing doesn't start.
⚠
Some Android phones limit USB OTG power. If the ESP32 disconnects mid-flash, power the ESP32 separately via its 5V pin while connecting data lines through OTG.

Step 2 · Choose Your Tool
1
Download Espressif Flash Download Tool
▶ espressif.com → Other Tools → Flash Download Tool
No install needed — just unzip and run flash_download_tool_x.x.x.exe
2
Install CH340 driver if needed
If your COM port doesn't appear: ▶ Download CH340 driver (WCH official)
3
Configure the tool
Open the tool → select ESP32 → UART. Add 3 files with these addresses:
@ 0x1000 → nRFBox_V2.ino.bootloader.bin @ 0x8000 → nRFBox_V2.ino.partitions.bin @ 0x10000 → nRFBox_V2.ino.bin
4
Select COM port and flash
Set COM port (check Device Manager), baud 460800, SPI speed 40MHz, SPI mode DIO. Click START. Hold BOOT if it doesn't begin.
1
Install Python + esptool
▶ Download Python (check "Add to PATH" during install), then:
pip install esptool
2
Find your COM port
Open Device Manager → Ports (COM & LPT) → look for CH340 or CP2102. Note the COM number e.g. COM5.
3
Flash — run in the folder with the .bin files
python -m esptool --chip esp32 --port COM5 --baud 460800 ^ write_flash ^ 0x1000 nRFBox_V2.ino.bootloader.bin ^ 0x8000 nRFBox_V2.ino.partitions.bin ^ 0x10000 nRFBox_V2.ino.bin

Step 2 · Flash with esptool

Install esptool via pip

2
Find your serial port
ls /dev/tty.* | grep -i usb
Look for /dev/tty.usbserial-xxxxx or /dev/tty.SLAB_USBtoUART
If nothing shows: ▶ Install CH340 driver for macOS
3
Flash — run in the folder with the .bin files
python3 -m esptool --chip esp32 \ --port /dev/tty.usbserial-0001 \ --baud 460800 write_flash \ 0x1000 nRFBox_V2.ino.bootloader.bin \ 0x8000 nRFBox_V2.ino.partitions.bin \ 0x10000 nRFBox_V2.ino.bin

1
Install esptool

2
Add yourself to dialout group (one-time)
sudo usermod -aG dialout $USER # log out and back in after this
3
Find your port
ls /dev/ttyUSB* /dev/ttyACM*
Usually /dev/ttyUSB0 for CH340 boards.
4
Flash
python3 -m esptool --chip esp32 \ --port /dev/ttyUSB0 \ --baud 460800 write_flash \ 0x1000 nRFBox_V2.ino.bootloader.bin \ 0x8000 nRFBox_V2.ino.partitions.bin \ 0x10000 nRFBox_V2.ino.bin
